Music Library Management System

This is a basic Java object-oriented programming project. It models a simple music library with classes for songs and albums.

Features
- Define songs with titles, artists, and durations.
- Group songs into albums.
- Demonstrate basic OOP concepts like encapsulation and composition.

How to Run
1. Clone the repository.
2. Compile the Java files using `javac`.
3. Run the program with `java MusicLibraryMain`.

 Future Improvements
- Add user input to create songs and albums.
- Save and load the library from a file or database.
- Implement search and filtering options.
